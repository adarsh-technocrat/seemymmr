Connect Google Search Console to see which search queries drive traffic and revenue to your website.

## Why connect Search Console?

By connecting Google Search Console, you can:

- See which search queries drive traffic
- Understand search performance metrics
- Link search queries to revenue
- Identify high-value search terms

## Setup

1. Go to your website settings in the dashboard
2. Click **"Integrations"** in the sidebar
3. Click **"Connect Google Search Console"**
4. Follow the OAuth flow to authorize PostMetric
5. Select the property you want to connect

## View search data

Once connected, you can view:

- **Search queries** - Which queries drive traffic
- **Click-through rate** - CTR for your search results
- **Average position** - Your average ranking position
- **Impressions** - How many times your pages appeared in search
- **Clicks** - How many clicks you received

## Link search to revenue

When Search Console is connected, you can see:

- Which search queries drive revenue
- Revenue by search query
- Conversion rate by search query

## Data sync

Search Console data is synced daily. You can also manually refresh the data from your dashboard.

## Privacy

PostMetric only accesses the Search Console data you authorize. We don't access any other Google services or data.

## Troubleshooting

### Can't connect Search Console

- Make sure you're logged into the correct Google account
- Verify you have access to the Search Console property
- Check that the property is verified in Search Console

### Data not syncing

- Wait 24 hours for initial sync
- Manually refresh the data from your dashboard
- Check that the property is still connected
